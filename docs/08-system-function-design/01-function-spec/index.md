# 機能定義書

=== "本文"

    機能定義書は、機能ごとにフォルダを分けて作成します。各機能フォルダの構成は共通で、
    以下のページを含みます。

    - 表紙
    - 画面設計
    - チェック処理概要
    - ファイル出力仕様
    - 更新条件表
    - 検索条件表

    [機能A](01-function-a/01-cover.md) を構成のテンプレートとして用意しています。
    機能を追加する場合は `01-function-a` フォルダを丸ごと複製し、フォルダ名と各ページの
    内容を対象の機能に合わせて変更してください。
